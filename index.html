¿Qué es GitHub?

GitHub es un sitio "social coding". Que te permite subir repositorios de código para almacenarlo en el sistema de control de versiones Git.

GitHub es una herramienta esencial para los ingenieros de software. Actualmente cuenta con más de 25 millones de usuarios. Se trata de un número considerable de profesionales que recurren a GitHub para mejorar el flujo de trabajo y la colaboración.

¿Qué es GIT?

Git es un sistema de control de versiones, diseñado por Linus Torvalds, que nos permitirá trabajar de manera colaborativa, haciendo fácil y óptimo el trabajo en equipo cuando estamos desarrollando software.
